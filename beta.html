Deck Standards

The standard deck contains 55 time bonuses, 24 curses, and 21 power-ups for a total of 100 cards. This doesn't need to be followed for custom decks, but can be helpful to know for game balancing.


Starting the Game

You will need to add 100 cards to your deck to start the game. You may use the "Add Base Deck" button to achieve this, or build a custom card that meets the 100 card minimum.
